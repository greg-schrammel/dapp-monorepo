# Tooltip

Default already styled tooltip used throughout the app  
based on `@radix-ui/react-tooltip`

```tsx
import { Tooltip } from 'ui'
```

```tsx live=true
<Tooltip content="this is a tooltip">
    <Button variant="icon" size="sm">
        <QuestionMarkIcon title="question mark" />
    </Button>
</Tooltip>
```
